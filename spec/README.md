# サンプルアプリケーション仕様

## 概要

iOS/Swift/SwiftUIを使用したサンプルアプリケーションを開発し、ネイティブ開発における基本的なアーキテクチャ構築と状態管理の手法を学習する。

## 学習目標

- SwiftUIによるUI構築の基礎
- MVVMアーキテクチャパターンの実装
- 状態管理（@State, @Binding, @ObservableObject, @EnvironmentObject）
- データの永続化（UserDefaults, SwiftData）
- 非同期処理（async/await）

## アプリ概要：タスク管理アプリ（Todo App）

シンプルなタスク管理アプリを題材として、上記の学習目標を達成する。

### 機能要件

| 機能 | 説明 |
|------|------|
| タスク一覧表示 | 登録済みタスクをリスト形式で表示 |
| タスク追加 | 新しいタスクを作成 |
| タスク編集 | 既存タスクのタイトル・詳細を編集 |
| タスク削除 | タスクを削除（スワイプ削除対応） |
| 完了状態の切り替え | タスクの完了/未完了を切り替え |
| データ永続化 | アプリ終了後もデータを保持 |

### 非機能要件

- 対応OS: iOS 17.0+
- 言語: Swift 5.9+
- UI: SwiftUI
- アーキテクチャ: MVVM
- データ永続化: SwiftData

## マイルストーン

1. **Phase 1**: プロジェクトセットアップ & 基本UI
   - Xcodeプロジェクト作成
   - 基本的なリスト表示

2. **Phase 2**: MVVM実装
   - Model定義
   - ViewModel実装
   - View-ViewModel連携

3. **Phase 3**: 状態管理
   - 各種Property Wrapperの活用
   - 画面間データ共有

4. **Phase 4**: データ永続化
   - SwiftData導入
   - CRUD操作の実装

5. **Phase 5**: 仕上げ
   - UIブラッシュアップ
   - エラーハンドリング
